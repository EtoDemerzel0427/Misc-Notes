# Windows Common Shortcuts

Here I only list those are useful ,common (but not that common) shortcuts.

## Normal

| shortcuts                                                    | ops                                                          |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| `Alt + Tab`                                                  | switch from opened apps                                      |
| `Win+L`                                                      | lock your PC                                                 |
| `Win+D`(or three fingers move inside/outside on the touchpad) | show or hide your desktop                                    |
| `F2`                                                         | rename chosen file                                           |
| `Ctrl+D`                                                     | delete chose files                                           |
| `Ctrl + Alt + Tab`                                           | switch from opened apps(Same as `Alt+Tab`?)                  |
| `Ctrl + Esc`                                                 | open START                                                   |
| `Ctrl + Shift + Esc`                                         | open task manager                                            |
| `Shift + Delete`                                             | directly delete a file without putting it to the Recycle bin |
| `Win + M`                                                    | minimize all                                                 |
| `Win + Shift + M`                                            | reverse of `Win + M`                                         |
